# HandTrackingMouse

## 概要
PCのマウス等の物理デバイスを使用せず、Webカメラで取得したユーザーの手の動きを用いて、画面上のポインタ操作およびクリック操作を実現する。
アクセシビリティの向上や、手が汚れている作業中などのハンズフリー操作を可能にする。

## 目的
本プロジェクトの目的は以下の通り。
- Webカメラ映像を低負荷で取得・解析する仕組みの構築
- 手検出・ランドマーク抽出によるポインタ操作の実装
- ジェスチャー認識の実装
- Rustによる軽量・安定した常駐アプリケーション開発

## 動作環境
- OS: Windows 11
- コンパイラ: rustc 1.94.0
- Rust edition: 2024
